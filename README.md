# 黑洞戰爭 — 直角坐標平面數位桌遊

國中七年級數學「直角坐標平面」單元的遊戲化教學工具。

## 功能

1. **遊戲規則** — 完整圖文說明（道具、卡牌、勝負判定）
2. **自動分組** — 輸入人數，隨機分為 3 人一組（2 人對戰 + 1 人監督）
3. **QR Code** — 每組自動配對遊戲房間，支援列印
4. **學習單** — 結合遊戲情境的 PDF 學習單，一鍵下載
5. **示範影片** — 嵌入 YouTube 完整示範

## 遊戲平台

使用 [playingcards.io](https://playingcards.io) 線上桌遊平台。
